Ship requires ALL of:
1. ☐ v1.7.0 runs clean on the live NAS (real 70-ZIM load), `/search` works
2. ☐ Eric has personally walked every core flow on the live box
3. ☐ BT/mDNS proven across 2 machines (discovery → torrent download → seed)
4. ☐ A11y verified on live (VoiceOver/Tab + Lighthouse ≥90)
5. ☐ Stale threads drained, issue replies drafted

No tag until 1–5 are checked with evidence (screenshot, command output, or Eric's explicit "saw it work").
